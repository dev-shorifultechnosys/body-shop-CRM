# Functional route map

| Route | Main interactions |
|---|---|
| Dashboard | KPIs, workflow cards, repair board, alerts, capacity, quick actions |
| Customers | Search, stage filter, add customer, open customer, message |
| Jobs & Repairs | Board/list toggle, drag stages, new repair order, export |
| Schedule | Week/day/capacity toggle, create appointment, print |
| Estimates | Search, filter, create/update estimates, CSV export |
| Sales Pipeline | Revenue chart, funnel, opportunities, CSV export |
| Invoices & Payments | Invoice list, invoice detail, record payment, export |
| Messages | Select conversations, send message, open customer |
| Reviews | Review metrics, request/reply actions |
| Vehicles | Vehicle search, VIN lookup, add vehicle |
| Parts & Inventory | Inventory metrics, add/manage parts, purchase order |
| Reports | Search reports, open analytics, build report, create folder |
| Team | Users, workload, permissions, invitations |
| Integrations | Connection cards and simulated management |
| Settings | Profile, shop, hours, permissions, notifications, appearance, setup |
| Help | Guides and support request |
